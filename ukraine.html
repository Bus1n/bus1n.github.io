Our approach

We asked girls aged 12-18 years old to express their fears, frustrations and needs to better understand how we can prevent further trauma and disempowerment.

Today, Ukraine is at a crossroads.

Why education for girls?

In Ukraine, only 23% of female university applicants choose to pursue careers in science, technology, engineering, or mathematics.

Closing the gender gap in digital fluency can open up new opportunities for young women and generate growth across the economy.

For example, studies have shown that increasing women's digital fluency can add $700 billion to the global economy by 2025.

Meet our support

Empowerment Lab offers live online courses with a teacher for girls aged 12-18 years old living in Ukraine.

offered

App Prototyping

E-COMMERCE MOBILE APP

Build your own e-commerce mobile app without any coding experience necessary! Create mood boards, extensive yet simplistic logo designs, color palettes and be able to engage thousands of users with your user experience flow & interface.

Skills developed:

  • Creative thinking
  • User research
  • Visual communication
  • Mobile app prototyping – Marvel app
  • Color palette design – Coolors software
  • Logo design – Looka & Canva softwares
By the end of the course, each student will have made their own personalized mobile app!

Machine Learning

THE BEAUTY ROBOT OF TOMORROW

Advance the way you experience shopping online. Map cosmetic items based on their ingredients similarities and give content-based filtering recommendations. No previous background in math or programming is required.

Skills developed:

  • Problem solving
  • Big picture thinking
  • Communication
  • Data Structuring - Google Colab
  • Programming language - Python- Web
  • Data Analysis - Python
By the end of this course, each student develops a simple cosmetics recommendation system.

Data-Driven Web App

SOCIAL SCIENCE WIZ UNITE

This cognitive science course allows you to engage concept values, get a better understanding of yourself and your friends, and use ORACLE Apex Software to spark a low code application platform to build a friendship app!

Skills developed:

  • Attention to detail
  • Critical thinking
  • Problem solving
  • Data Analytics & Visualization - Oracle APEX
  • Database Design - SQL
  • Database Programming - SQL
By the end of the course, each student develops a customized, hypothesis-driven web application that enables interactive data analysis.

Data Analysis in Excel

DATA ANALYSIS OF
ENDANGERED SPECIES

A beginner-level introductory course for gaining awareness of social causes and animal activism! This course helps you understand how data can help make insightful observations and judgments on our global atmosphere while mastering the advantages of data analytic programs like Microsoft Excel.

Skills developed:

  • Draw parallels to world problems
  • Analytical thinking
  • Public speaking
  • Data research
  • Data analysis - Microsoft Excel
  • Data visualisation - Microsoft Excel
By the end of this course, you will build an interactive presentation with your own findings and recommendations.

Digital Media

TELL A STORY. BE A LEGEND

In this Digital Media based course, you will gain the insight and initiatives to discover storytelling through a new lens. Whether you are just starting or have been a seasoned amateur photographer, this course will help you expand your horizons.

Skills developed:

  • Critical thinking
  • Public speaking
  • Photo editing - Pixlr software
  • Web Design - WordPress software
  • Web Development - WordPress software
By the end of this course, you will have a full- functioning live website.

Game Development

ONCE UPON A PIXEL TIME

The Video Gaming Industry is taking over. It’s now offering thousands of jobs all catered to creative storytelling and exploring your wildest imagination. Choose an adventure, pick a path, create a character, and tell a story. Learn how to get paid to have fun.
